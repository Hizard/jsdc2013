Slides for JSDC 2013. 

Real-Time Web Application with Node.js, Socket.io, and Redis.

1. Install dependencies
```
npm install
```

2. Start Redis
```
download redis
make
src/redis-server
```

3. Start Node.js
```
npm start
```

Done. Connect to

http://localhost:3939/

To edit the code
```
coffee -o build -cw src
```
